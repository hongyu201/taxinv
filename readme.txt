脚本功能：
概要：通过国家税务局发票查验平台查验发票真伪
1， 通过扫描枪扫描发票二维码自动获取发票信息（发票代码、发票编号、开票日期、开票金额、验证码等）
2， 手动输入验证码，查验发票
3， 自动保存查验结果到PDF文件， 文件名为发票代码
4， 自动开始下一张发票查询

执行环节说明：
chrome  78.0.3904.108 + chromedriver 78.0.3904.105
firefox50.1 + geckodriver 0.16.1 功能限制很多，尝试多次后放弃；
firefox55 + geckodriver 0.19 可正常打开网页，查找元素出错，放弃；
最终选择为chrome78.0.3904.108 + chromedriver 78.0.3904.105
